Berks Career and Technology Center Student Organizations

The CTSO Website is for students and organizers to learn more about their extra corricular group. This allows organizers to post updates about their group. Some of this information includes what the program is about and current Events. The most important feature is the Calander. When the user clicks on a caladar button that contains an event a pop up is shown. This pop up shows name, a small description, and a link to a full description tab. The Next tab is more descriptive showing location, time, and additional files.
